== Description ==

**Advanced Product Filters for WooCommerce** enhances your WooCommerce store by providing custom filtering capabilities. Customers can filter products based on criteria like price, category, stock status, sale status, tags, rating, and product attributes.

### Key Features:
– **Custom Filters**: Easily add filters for various product attributes, such as price range, category, and stock status.
– **Enable/Disable Filters**: Manage the visibility of filters on the frontend through the plugin settings.
– **Nested Categories**: Display nested categories with product counts for a better user experience.
– **Custom Styles and Scripts**: Customize the appearance and behavior of filters with your own CSS and JavaScript.

== Installation ==

1. **Upload the Plugin**  
   – Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin via the WordPress plugins screen directly.

2. **Activate the Plugin**  
   – Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

1. Go to **WooCommerce > Advanced Filters Settings** in the admin area.
2. **Enable or Disable Filters**:
   – Check the boxes for the filters you want to show.
3. Visit the shop or category pages to see the filters in action.

== Configuration ==

### Settings Page
1. Go to **WooCommerce > Advanced Filters Settings**.
2. Configure filter options such as:
   – **Product Name**: Toggle visibility for product name searches.
   – **Price Range**: Enable or disable price filtering.
   – **Category**: Show categories with counts.
   – **Stock Status**: Filter by stock status.
   – **On Sale**: Include a filter for sale products.
   – **Tags**: Add tag-based filters.
   – **Rating**: Enable rating-based filters.
   – **Product Attributes**: Select attributes for filtering.

### Custom Styles and Scripts
– Customize styles in the `css/style.css` file.
– Add JavaScript in the `js/scripts.js` file.
